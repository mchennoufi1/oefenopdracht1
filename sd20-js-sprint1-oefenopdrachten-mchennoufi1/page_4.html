Functions

Oke je bent echt lekker bezig!! Tijd voor de volgende stap. Functions in JavaScript.

Hebben we je net alles geleerd in C# over methods, gaan we het nu hebben over functions.
MAAR! goed nieuws. Het is gewoon hetzelfde :-D het heet alleen iets anders.

functions in JavaScript

Om een method aan te maken in JS gebruik je standaard het woord function

In JS geef je niet op wat een function teruggeeft. In C# moet dit wel, bijvoorbeeld:

public int GetAge()

In JS doen we dit niet. Dezelfde functie in JS is dan dus:

function getAge()

OMG het is allemaal anders

Oke rustig, rustig :-) Het lijkt allemaal anders, maar dat is niet waar!

Maar je moet wel lezen

C# en JavaScript lijken heel erg op elkaar. Ze zijn allebei afgeleiden van Java en C++. De syntax (hoe je de code schrijft) lijkt daarom erg op elkaar. PHP (dit krijg je in het tweede jaar) is ook een afgeleide van die talen, daarom lijken ze best wel op elkaar.

Code Convention

Toch zijn er verschillen, maar dat valt mee, en gaan we je uiteraard bij helpen.
Code Convention is een een lijst van afspraken hoe je omgaat met de code. Hoe werken methods (functions) wat doe je met variabele en zo nog veel meer.
